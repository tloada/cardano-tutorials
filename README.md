# Tutoriales

Aquí podés encontrar una series de recursos para ayudarte en tu instalación, operación y desarrollo frente a la plataforma de Cardano. Constantemente estmos desarrollando material para ayudarte comprender nuestras plataforma y utilizar sus funciones. Estos tutoriales fueron traducidos por  **stake pool THE LEGEND OF ₳DA [TLo₳]**.

La edición actual de tutoriales se concentra en preparar y monitorear el nodo y el uso de la Interfaz de Línea de Comando (CLI) para realizar un sinnúmero de operaciones fundamentales. Los tutoriales se encuentran [aquí](iniciando-nodo/).
